# Project Technologies

This project is built using the following technologies and libraries:

## Core Framework
- **[React](https://react.dev/)** (v19.x): A JavaScript library for building user interfaces.
- **[Vite](https://vitejs.dev/)**: Next Generation Frontend Tooling for fast serving and building.

## Styling
- **[Tailwind CSS](https://tailwindcss.com/)** (v3.x): A utility-first CSS framework for rapid UI development.
- **PostCSS**: A tool for transforming CSS with JavaScript.
- **Autoprefixer**: PostCSS plugin to parse CSS and add vendor prefixes to CSS rules.

## Animations & Interactive UI
- **[Framer Motion](https://www.framer.com/motion/)**: A production-ready motion library for React, used for the hero section and element transitions.
- **[Swiper](https://swiperjs.com/)**: Most modern mobile touch slider, used for the "Gaming Projects", "Product Animation", and "Gallery" carousels.
- **[Lucide React](https://lucide.dev/)**: Flexible icon family for the interface icons.

## Routing
- **[React Router DOM](https://reactrouter.com/)**: Standard routing library for keeping the UI in sync with the URL.

## Utilities
- **clsx**: A tiny utility for constructing `className` strings conditionally.
- **tailwind-merge**: Utility function to efficiently merge Tailwind CSS classes without conflicts.

## Development & Linting
- **ESLint**: For identifying and reporting on patterns found in ECMAScript/JavaScript code.
